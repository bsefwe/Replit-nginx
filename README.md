```
传输层安全：tls
传输协议：WebSocket
WebSocket 路径：/vless 或 /vmess
```

### 自定义 UUID

Replit 内点击  🔒Secrets 选项

设定 ```key``` 为 ```UUID```
设定 ```value``` 为 ```UUID 内容```

点击 Add new secret 保存

重启应用生效
